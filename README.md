# Coordinatics APP

APP de la prueba de Coordinatics.

# Instalación

  - Clonar repositorio.
  - Ejecutar el siguiente comando:
    ```sh
    $ yarn install (o npm install)
    ```
# Uso

  - Ejecutar el siguiente comando:
    ```sh
    $ yarn start (o npm start)
    
# Nota
Para efectos de la prueba, se utilizó el instalador de paquetes Yarn.
